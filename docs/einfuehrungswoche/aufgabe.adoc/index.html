August 2018 :page-license: {cc-by-sa} :toc2: {empty} == Vorbereitung Um https://processing.org/download/[Processing] zu installieren, als Entwicklungsumgebung kennen zu lernen und auch erste Programme zu schreiben -- diese werden in Processing Sketches genannt --, folgen Sie dem einführenden Abschnitt http://michaelkipp.de/processing/02%20processing.html[Processing - Entwicklungsumgebung] von Michael Kipp. Alle Befehle sind in der englischsprachigen https://processing.org/reference/[Processing Language Referenz (API)] dokumentiert und werden dort auch anhand von Beispielen erläutert. Es werden Aufgaben auf verschiedenen Schwierigkeitsniveaus angeboten. Wählen Sie entsprechend Ihrer Vorkenntnisse das passende aus. == Niveau 1 -- Statische Grafiken -- keine Vorkenntnisse Mit Processing können statische Grafiken erstellt werden.
