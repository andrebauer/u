1. Vorbereitung

Um Processing zu installieren, als Entwicklungsumgebung kennen zu lernen und auch erste Programme zu schreiben — diese werden in Processing Sketches genannt --, folgen Sie dem einführenden Abschnitt Processing - Entwicklungsumgebung von Michael Kipp.

Alle Befehle sind in der englischsprachigen Processing Language Referenz (API) dokumentiert und werden dort auch anhand von Beispielen erläutert.

Es werden Aufgaben auf verschiedenen Schwierigkeitsniveaus angeboten. Wählen Sie entsprechend Ihrer Vorkenntnisse das passende aus.

2. Niveau 1

Statische Grafiken — keine Vorkenntnisse

Mit Processing können statische Grafiken erstellt werden. Die Grafik wird entsprechend der Reihenfolge der Grafikbefehle erstellt.

Beispiel

size(300,300);
background(173,204,214);
fill(49,87,100);
strokeWeight(5);
strokeJoin(ROUND);
stroke(34,60,70);
line(50,40,100,280);
rect(30,70,60,30);
fill(152,64,72);
ellipse(100,200,100,70);
strokeWeight(8);
point(200,250);
noStroke();
arc(170,120,200,200,radians(-125),radians(35));
fill(52,180,27);
textSize(50);
text("Processing", 10, 180);

3. Niveau 2

Interaktive Grafiken — Vorkenntnisse mit Grafikprogrammierung

Die folgenden Sketches sind interaktiv und reagieren auf die Position des Mauszeigers. Interaktive Sketches bestehen zumindest aus den beiden Funktionen setup() und draw(). Die Funktion setup() wird beim Start genau einmal ausgeführt, die Funktion draw() wird nach setup() bis zum Programmabbruch wiederholt („endlos“) ausgeführt.

Beispiel

void setup() {
  size(300,300);
  noStroke();
}

void draw() {
  background(173,204,214);
  fill(49,87,100);
  if (mouseX > 0 && mouseX < 100) {
    rect(0,0,100,300);
  }
  if (mouseX >= 100 && mouseX < 200) {
    rect(100,0,100,300);
  }
  if (mouseX >= 200 && mouseX < 300) {
    rect(200,0,100,300);
  }
}

4. Niveau 3

Zustandsbasierte, interaktive Sketches — Vorkenntnisse mit Grafikprogrammierung und bedingten Anweisungen

Viele Sketches können nur mithilfe von Variablen realisiert werden, die den Systemzustand abbilden. Die Funktion mousePressed() wird vom System ausgeführt, wenn die Maus gedrückt wurde. Im Beispiel ändert diese den Zustand der Variablen bgcolor.

Beispiele

int bgcolor = "blue";

void setup() {
  size(300,300);
}

void draw() {
  if (bgcolor == "blue") {
    background(173,204,214);
  } else if (bgcolor == "red") {
    background(152,64,72);
  }
}

void mousePressed() {
  if (bgcolor == "blue") {
    bgcolor = "red";
  } else {
    bgcolor = "blue";
  }
}

int x, y, dx, dy, r;

void setup() {
  x = 125;
  y = 10;
  dx = 0;
  dy = 11;
  r = 20;
  size(300,300);
}

void draw() {
  background(173,204,214);
  fill(49,87,100);
  ellipse(x, y, 2*r, 2*r);
  x = x + dx;
  y = y + dy;
  if (y > height || y < 0) {
    dy = -1 * dy;
  }
}

5. Niveau 4

Vorkenntnisse in Datenstrukturen, Variablen und Grafik-Programmierung — Vorkenntnisse mit zustandsbasierter Programmierung und interaktiven Sketchen

Für dieses Niveau werden keine vorgegebenen Sketches als Aufgaben vorgegeben, sondern es werden lediglich Ideen genannt, wie der Sketch mit dem abprallenden Ball von Niveau 3 weiterentwickelt werden kann.

Mögliche Aufgaben

  • Erstellen Sie einen Sketch mit mehreren abprallenden Bällen

  • Fügen Sie eine Möglichkeit hinzu, dass neue Bälle per Mausklick hinzugefügt werden können.

  • Erstellen Sie ein Pong-Spiel: Ergänzen Sie den abprallen Ball um ein steuerbares Panel, an dem der Ball ebenfalls abprallt.
