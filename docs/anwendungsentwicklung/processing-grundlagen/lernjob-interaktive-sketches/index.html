Lernziele

Zusätzlich zu den Zielen des Lernjobs Statische Sketches:

  • Ich habe dynamische, iteraktive Sketches erstellt und dazu die Funktionen setup() und draw() gezielt eingesetzt.

  • Ich habe globale Variablen wie z. B. mouseX, mouseY und mousePressed für die Interaktion mit dem Bentuzer eingesetzt und dazu deren Bedeutung in der Processing-Referenz nachgeschaut.

  • Ich habe mit bedingten Anweisungen (if-Anweisungen) den Programmablauf gesteuert.

Beispiel

Interaktive Sketches bestehen zumindest aus den beiden Funktionen setup() und draw(). Die Funktion setup() wird beim Start genau einmal ausgeführt, die Funktion draw() wird nach setup() bis zum Programmabbruch wiederholt („endlos“) ausgeführt.

void setup() {
  size(300,300);
  noStroke();
}

void draw() {
  background(173,204,214);
  fill(49,87,100);
  if (mouseX > 0 && mouseX < 100) {
    rect(0,0,100,300);
  }
  if (mouseX >= 100 && mouseX < 200) {
    rect(100,0,100,300);
  }
  if (mouseX >= 200 && mouseX < 300) {
    rect(200,0,100,300);
  }
}
