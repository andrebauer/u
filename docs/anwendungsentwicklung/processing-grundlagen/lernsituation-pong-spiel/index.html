Beispiel

Kollisionserkennung

int r1, r2;

void setup() {
  size(300,300);
  r1 = 30;
  r2 = 25;
}

/**
 * Gib genau dann den Wert true zurück,
 * sich zwei Kreise mit den
 * Mittelpunkten M1(x1, y1) und M2(x2,y2)
 * und Radien r1 und r2 überlappen.
 */
boolean overlap(int x1, int y1, int x2, int y2, int r1, int r2) {
  boolean overlaps = dist(x1,y1,x2,y2) <= r1 + r2;
  return overlaps;
}

void draw() {
  background(173,204,214);
  fill(49,87,100);
  if (overlap(150,150,mouseX,mouseY,r1,r2)) {
    fill(152,64,72);
  }
  ellipse(150,150,2*r1,2*r1);
  ellipse(mouseX,mouseY,2*r2,2*r2);
}

Aufgabe

Stellen Sie Ihr Ergbnis als pde-Datei in Ihrem E-Portfolio zur Verfügung.

Erstellen Sie ein Pong-Spiel: Ergänzen Sie den Sketch „Abprallender Ball“ um ein steuerbares Panel, an dem der Ball ebenfalls abprallt.


Zum Neustart drücke eine Maus-Taste.
