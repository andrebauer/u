Ziele

Zusätzlich zu den Zielen der Lernjobs Statische Sketches, Interaktive Sketches und Zustandsbasierte interaktive Sketches:

  • Ich habe mithilfe von Arrays oder Listen den Zustand von mehreren (graphischen) Objekte abgebildet.

  • Ich habe mithilfe von for-Schleifen den Zustand von mehreren (graphischen) Objekten ausgelesen und/oder verändert.

  • Ich habe auf einzelne Elemente von Arrays oder Listen zugegriffen.

  • Ich habe mein Programm zur Stukturierung in mehrere Funktionen aufgeteilt.

Beispiele

Statisches Säulendiagramm

Die Funktion bars(left, bottom, barwidth, margin, heights) zeichnet ein Säulendiagramm.

int[] barHeights = new int[]{10, 40, 80, 105, 125, 120, 95, 55, 20, 12};

void setup() {
  size(300,200);
  background(173,204,214);
}

void bars(int left, int bottom, int barwidth, int margin, int[] heights) {
  int x = left;
   for(int i = 0; i < heights.length; i++) {
     rect(x, bottom - heights[i], barwidth, heights[i]);
     x = x + barwidth + margin;
   }
}

void draw() {
  fill(49,87,100);
  bars(10,190,20,9,barHeights);
}

Dynamische Säulendiagramm

int[] barHeights = new int[]{10, 40, 80, 105, 125, 120, 95, 55, 20, 12};

void setup() {
  size(300,200);
  background(173,204,214);
  frameRate(15);
}

void bars(int left, int bottom, int barwidth, int margin, int[] heights) {
  int x = left;
   for(int i = 0; i < heights.length; i++) {
     rect(x, bottom - heights[i], barwidth, heights[i]);
     x = x + barwidth + margin;
   }
}

void rotate(int[] array) {
  int left = array[0];
  for(int i = 0; i < array.length - 1; i++) {
    array[i] = array[i + 1];
  }
  array[array.length - 1] = left;
}

void draw() {
  background(173,204,214);
  fill(49,87,100);
  bars(10,190,20,9,barHeights);
  rotate(barHeights);
}
