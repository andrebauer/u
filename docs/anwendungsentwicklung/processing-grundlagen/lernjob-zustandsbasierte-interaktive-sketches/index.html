Viele Sketches können nur mithilfe von Variablen realisiert werden, die den Systemzustand abbilden.

Lernziele

Zusätzlich zu den Zielen der Lernjobs Statische Sketches und Interaktive Sketches:

  • Ich habe Methoden zur Benutzerinteraktion wie z.B. mousePressed() oder keyPressed() eingesetzt und dazu deren Bedeutung und Verwendung in der Processing-Language-Reference nachgeschaut.

  • Ich habe Variablen eingesetzt, um den Zustand des Systems abzubilden, und deren Werte gesetzt, geändert und ausgelesen.

  • Ich habe mithilfe von Variablen und bedingten Anweisungen (if-Anweisung, else if und else) den Programmablauf gesteuert.

  • Ich habe globale Variablen wie height und width verwendet, damit meine Sketches auch nach einer Größenveränderung mit size noch sinnvoll funktionieren.

  • Ich habe Basisdatentypen wie int, String, boolean und double für Variablen ausgewählt und eingesetzt.

Beispiele

Die Funktion mousePressed() wird vom System ausgeführt, wenn die Maus gedrückt wurde. Im Beispiel ändert diese den Zustand der Variablen bgcolor.

Farbwechsel

String bgcolor = "blue";

void setup() {
  size(300,300);
}

void draw() {
  if (bgcolor == "blue") {
    background(173,204,214);
  } else if (bgcolor == "red") {
    background(152,64,72);
  }
}

void mousePressed() {
  if (bgcolor == "blue") {
    bgcolor = "red";
  } else {
    bgcolor = "blue";
  }
}

Abprallender Ball

Die Variablen x und y beinhalten die aktuelle Position des Balls, die Variablen dx und dy die Bewegungsrichtung. Die Variable r steht für den Radius des Balls.

int x, y, dx, dy, r;

void setup() {
  x = 125;
  y = 10;
  dx = 0;
  dy = 11;
  r = 20;
  size(300,300);
  background(173,204,214);
}

void draw() {
  background(173,204,214);
  fill(49,87,100);
  ellipse(x, y, 2*r, 2*r);
  x = x + dx;
  y = y + dy;
  if (y > height || y < 0) {
    dy = -1 * dy;
  }
}
