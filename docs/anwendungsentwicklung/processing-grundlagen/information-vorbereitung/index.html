Einstieg in Processing

Um Processing zu installieren, als Entwicklungsumgebung kennen zu lernen und auch erste Programme zu schreiben, diese werden in Processing Sketches genannt, folgen Sie dem einführenden Abschnitt Processing - Entwicklungsumgebung von Michael Kipp.

Alle Befehle sind in der englischsprachigen Processing Language Referenz (API) dokumentiert und werden dort auch anhand von Beispielen erläutert.
